
    ```json
      "error": "Không có dữ liệu để import"
    }
    ```

    ```javascript
    if (!row.name)
    ```

    ```

    ```

    → Không insert bản ghi đó.

  * Thực hiện insert:

    ```sql
    INSERT INTO products
    (name, price, description, imageUrl, category_id)
    ```

    ```javascript
    row.description || ""
    ```

    → description mặc định = ""

    ```javascript

    ```

    → imageUrl mặc định = ""

    ```javascript
    row.category_id || 1
    ```

    → category_id mặc định = 1


## 4. AI Gap Analysis & Vulnerability Report

### 4.1 Specification Gaps

| Gap ID | Description | Affected Variables | Risk Level | Recommendation |
|--------|-------------|-------------------|------------|----------------|
| GAP-001 | **No role/permission check confirmed** — `/api/admin/` path suggests admin-only, but no `role === 'admin'` guard found. Regular users may be able to import products. | `Authorization` header | **CRITICAL** | Add middleware: `if (req.user.role !== 'admin') return res.status(403).json({error: "Forbidden"})` |
| GAP-002 | **`category_id: 0` silently overrides to 1** — `0 \|\| 1` evaluates to `1` due to JS falsy coercion. Intent ambiguous: user supplied a value but it gets silently changed. | `category_id` | **HIGH** | Replace `row.category_id \|\| 1` with `row.category_id !== undefined && row.category_id !== null ? row.category_id : 1` |
| GAP-003 | **No `price` validation** — negative, zero, float, null values all accepted without error. Negative-price products create financial logic errors. | `price` | **HIGH** | Add: `if (typeof row.price !== 'number' \|\| row.price < 0) → row error` |
| GAP-004 | **No `name` whitespace trim** — `"   "` (spaces-only) is truthy in JS, bypasses `!row.name` check, inserts a product with meaningless whitespace name | `name` | **HIGH** | Add: `if (!row.name \|\| !row.name.trim()) → row error` |
| GAP-005 | **No `category_id` foreign key existence check** — non-existent `category_id` values (e.g., 9999) may be stored silently, breaking category-product relations | `category_id` | **HIGH** | Validate FK before insert: `SELECT id FROM categories WHERE id = ?`; error if not found |
| GAP-006 | **No max length on `name`** — 1000-char product names may be stored, causing UI display issues and potential column overflow | `name` | **MEDIUM** | Cap at 255 chars: `if (row.name.length > 255) → row error "Hàng X: Tên sản phẩm quá dài"` |
| GAP-007 | **No `imageUrl` format validation** — `"javascript:alert(1)"` stored as imageUrl renders as executable XSS if displayed in `<img src>` | `imageUrl` | **HIGH** | Validate URL format: allow only `http://` and `https://` schemes; reject `javascript:`, `data:` |
| GAP-008 | **No duplicate product name detection** — concurrent or sequential imports with identical names create ghost duplicates in catalog | `name` | **MEDIUM** | Add optional unique check or duplicate-detection report in response `errors` array |
| GAP-009 | **No import size limit** — 10,000-row payloads accepted, risking server OOM / timeout / DB lock contention | `products.length` | **MEDIUM** | Add: `if (rows.length > 1000) return res.status(400).json({error: "Giới hạn 1000 sản phẩm mỗi lần import"})` |
| GAP-010 | **No transaction rollback on partial failure** — successful rows before a failed row are permanently committed. Partial imports may leave catalog in inconsistent state. | All row fields | **MEDIUM** | Wrap all inserts in a DB transaction; rollback all on any error, or document partial-commit behavior explicitly |
| GAP-011 | **`price` type not validated** — string `"150000"` may be coerced or stored as TEXT in SQLite, breaking numeric sorting/filtering | `price` | **MEDIUM** | Add: `if (typeof row.price !== 'number') → row error "Hàng X: Giá sản phẩm không hợp lệ"` |
| GAP-012 | **Row index convention (0-based vs 1-based) not specified** — error message `"Hàng X"` index derivation not documented; may differ from user expectation | Error messages | **LOW** | Document and test: first row = `"Hàng 1"` (1-based) vs `"Hàng 0"` (0-based) |
| GAP-013 | **No `price` required validation** — missing price field likely inserts NULL into `price` column; products without price cannot be sold | `price` | **MEDIUM** | Add: `if (row.price === undefined \|\| row.price === null) → row error "Hàng X: Thiếu giá sản phẩm"` |
